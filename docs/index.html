ANFCapacityManager

ANFCapacityManager is an Azure logic app that can automatically create metric alert rules in Azure Monitor to alert you when volumes are getting full. Optionally, it can also increase the volume sizes automatically to keep your applications online.

ANFHealthCheck

ANFHeathCheck is a PowerShell runbook that generates beautiful HTML reports of your entire Azure NetApp Files landscape. Optionally, it can automatically reduce over-sized volumes and capacity pools to reduce your TCO.
